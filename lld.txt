HLD (High-Level Design)
Functional Requirements

User should be able to select the city
User should be able to search and select restaurants
User should be able to select food items
User should be able to add items to cart
Prevent duplicate order submission
User must be able to make payment
User must get order & payment confirmation

Non-Functional Requirements

High Availability
Low Latency
Scalability
Secure Payment Processing
Fault Tolerance
